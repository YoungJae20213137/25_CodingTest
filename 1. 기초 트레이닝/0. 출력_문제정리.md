## 출력

### Q1(문자열 출력하기) / 출력
```Python
if __name__ == "__main__":
    str = input()
    print(str)
```
**주요 개념 및 로직**
1. str을 input()으로 받아, print()로 출력하기

### Q2(a와 b 출력하기) / 출력
```Python
if __name__ == "__main__":
    a, b = map(int, input().split())
    print(f"a = {a}\nb = {b}")
```
**주요 개념 및 로직**
1. a, b를 공백을 제거하고, int형으로 입력받기
2. format에 맞추어 출력하기

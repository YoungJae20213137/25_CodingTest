### Q96(배열의 길이를 2의 거듭제곱으로 만들기) / 함수(메서드)
```Python
def power2(arr):
    power = 1
    
    while (power < len(arr)):
        power *= 2
        
    return power

def solution(arr):
    answer = []
    answer = arr
    
    power_val = power2(arr)
    padding = power_val - len(arr)
    
    answer.extend([0] * padding)
        
    return answer

if __name__ == "__main__":
    arr = list(map(int, input()))
    
    result = solution(arr)
    print(result)
```
